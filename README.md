# Bearing Capacity Failure Modes — Interactive

地盤工学講義 b02 第8回「支持力（古典理論）」第3節「支持力破壊の3つの形態」のためのブラウザ完結インタラクティブ教材。Plotly.js + MathJax を CDN 読み込みするのみで、ビルド不要・依存ファイルなしの静的ページ。

## 教育目的

Vesic (1973) の分類による支持力破壊の3形態 — **全般せん断破壊 (general shear)**・**局部せん断破壊 (local shear)**・**パンチング破壊 (punching shear)** — の差異を、

- **荷重-沈下曲線の形**（ピークの有無・軟化）
- **すべり面の地表への到達状況**（地表隆起の有無）

の2つの観点で対比する。相対密度 $D_r$（地盤の密度・強度・剛性の代理指標）を動かすと、破壊形態が連続的に移り変わる。

| 形態 | 荷重-沈下曲線 | すべり面 | 地盤条件 |
|------|--------------|---------|---------|
| 全般せん断 | 明確なピーク後に急激な沈下 | 基礎下から地表まで到達・隆起 | 密な砂・硬い粘土 ($D_r \ge 70\%$) |
| 局部せん断 | ピークが不明瞭 | 基礎下に発達するが地表に達しない | 中位 ($D_r \approx 35\text{–}70\%$) |
| パンチング | ピークなし単調漸増 | 鉛直方向に貫入・隆起なし | 緩い砂・軟らかい粘土 ($D_r < 35\%$) |

## 操作

- **探索モード**: $D_r$ スライダーで形態が自動的に切り替わる。例題1 のプリセット (a)(b)(c) を用意。
- **比較モード**: 3形態の地盤断面を縦に並べて同時表示。
- **載荷アニメーション**: 載荷の進行に伴い、基礎の沈下・すべり面の発達・地表隆起が時間発展する。

## モデルの位置づけ

構成則（弾塑性 FEM 等）は解いていない。荷重-沈下曲線は各形態の定性的特徴を再現する解析的近似式、すべり面は主働くさび（$45^\circ+\phi'/2$）・対数らせん遷移帯・受働域（$45^\circ-\phi'/2$）の幾何を模式化したものである。$\phi'$ は $D_r$ から $\phi' \approx 28^\circ + 14^\circ D_r$ で推定する。定量値ではなく形態間の差異を体感することが目的。
